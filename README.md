# Game Landing — `game.vasilevagalina.ru`

Мобильный лендинг для записи на трансформационные игры (Лила + Игра про деньги).

## Структура

- `index.html` — разметка
- `styles.css` — стили (палитра и шрифты совпадают с `start.vasilevagalina.ru`)
- `script.js` — ссылки в мессенджеры, быстрый подбор игры, FAQ-accordion, Яндекс.Метрика
- `assets/` — фото ведущей, иконки игр, og-превью
- `CNAME` — поддомен для GitHub Pages

## Локальный предпросмотр

```bash
cd landing/game
python3 -m http.server 8000
# открыть http://localhost:8000 в Chrome DevTools → iPhone SE
```

## Куда вставлять данные от клиента

Параметры в `script.js` — в секции `CONFIG` вверху файла. Основные тексты, цены и FAQ уже зашиты в `index.html`.

### 1. Ссылки в бот (`script.js`, блок `CONFIG`)

```js
TELEGRAM_URL: "https://t.me/<bot>?start=game",
MAX_URL:      "<max-deeplink>",
```

### 2. Цены

Зашиты в `index.html` в блоке `<section id="pricing">`:
- Индивидуально (2,5 ч): онлайн 8 000 ₽ / офлайн 10 000 ₽
- Вдвоём (3 ч): онлайн 4 000 ₽ / офлайн 5 000 ₽

Одинаковы для Лилы и Игры про деньги.

### 3. Тексты (`index.html`)

Проверить перед релизом:
- регалии ведущей
- FAQ-ответы
- адрес офлайн-встречи
- формулировки стоимости и условий переноса

### 4. Фото (`assets/`)

- **`host.jpg`** — фото ведущей. HTML уже ссылается на `./assets/host.jpg`. Положить присланный файл сюда под этим именем. Желательно — вертикальный кадр (3:4), ширина ≥600px, оптимизированный (`<200 KB`, через tinypng.com или `cwebp`).
- `og.jpg` (опционально) — превью для шаринга (1200×630). Сейчас `og:image` временно ссылается на `host.jpg`, чтобы не было битой ссылки.
- Иконки игр `lila.svg` / `money.svg` — опционально, если клиент пришлёт.

## Деплой

Репозиторий подключается к GitHub Pages с `CNAME = game.vasilevagalina.ru`. Проверить, что A-записи / CNAME в DNS направлены на GitHub Pages.
